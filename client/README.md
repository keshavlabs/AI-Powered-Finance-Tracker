# AI Finance Tracker 💰

A full-stack AI-powered finance tracker built with:

## 🚀 Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- AI: Groq API

## ✨ Features

- User authentication
- Expense tracking
- AI financial assistant
- Chat-based insights
- Smart financial analysis

## 🛠️ Setup

### Backend

cd server  
npm install  
npm run dev

### Frontend

cd client  
npm install  
npm run dev
